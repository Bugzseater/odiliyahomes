# Project structure

- src/
  - assets/ — static assets colocated with code (images, fonts)
  - components/ — reusable UI building blocks
  - features/ — feature slices (domain-focused)
  - hooks/ — reusable React hooks
  - layouts/ — page layouts and shells
  - lib/ — shared libs, adapters (api, http, analytics)
  - pages/ — route-level pages
  - routes/ — router setup
  - services/ — API clients and business services
  - store/ — state management (Redux/Zustand/Context)
  - styles/ — global CSS, tokens, utilities
  - utils/ — pure helpers
